# 第1章：变量、数据类型、HelloWorld

## 1.1 理论：变量是什么

**Lua 的变量：**
```lua
name = "战士"    -- 字符串
level = 30       -- 数字
is_alive = true  -- 布尔
-- Lua 自动识别类型，随时可以改变
name = 100       -- 完全合法
```

**C++ 的变量（必须先声明类型）：**
```cpp
string name = "战士";   // 字符串
int level = 30;          // 整数
double hp = 100.0;       // 浮点数
bool is_alive = true;    // 布尔
```

**为什么 C++ 要声明类型？**
- 告诉计算机分配多少内存
- 编译时检查错误，运行前就发现
- 编译器优化，运行更快

---

## 1.2 理论：常用数据类型

| 类型 | 含义 | 举例 | 占用 |
|------|------|------|------|
| `int` | 整数 | `30`, `-10`, `0` | 4字节 |
| `double` | 浮点数 | `100.5`, `3.14` | 8字节 |
| `float` | 单精度浮点 | `172.5f` | 4字节 |
| `char` | 单个字符 | `'A'`, `'好'` | 1字节 |
| `bool` | 布尔 | `true`, `false` | 1字节 |
| `string` | 字符串 | `"战士"` | 动态 |

**和 Lua 的对应：**
```
Lua 类型        C++ 类型
────────────────────────
nil              无对应
number（整数）    int
number（小数）    double
string           string（需 #include <string>）
boolean          bool
table            vector + struct + map
```

## 1.3 理论：第一个 C++ 程序

```cpp
#include <iostream>      // 引入输入输出工具（相当于 Lua 的 require）
using namespace std;   // 使用标准命名空间

int main() {            // 程序入口，从这里开始执行
    cout << "Hello, World!" << endl;
    return 0;           // 返回 0 表示正常结束
}
```

**逐行解释：**
- `#include <iostream>`：把 iostream 文件内容复制进来，提供 `cout`、`cin`、`endl`
- `using namespace std;`：让 `cout` 不用写成 `std::cout`
- `int main() { }`：主函数，程序入口
- `cout << "Hello" << endl;`：打印内容到屏幕

# 第2章：条件判断

## 2.1 理论：if / else

**Lua：**
```lua
if hp <= 0 then
    print("死亡")
elseif hp < 30 then
    print("危险")
else
    print("正常")
end
```

**C++：**
```cpp
if (hp <= 0) {
    cout << "死亡" << endl;
} else if (hp < 30) {
    cout << "危险" << endl;
} else {
    cout << "正常" << endl;
}
```

**三个区别：**
- C++ 条件要加括号 `( )`
- C++ 代码块用 `{ }` 包裹
- Lua 用 `elseif`，C++ 用 `else if`（两个词）

---

## 2.2 理论：赋值和比较的区别

```cpp
int x = 10;

// 错误：= 是赋值，条件永远为 true！
if (x = 20) {
    cout << "x是20";
}

// 正确：== 才是比较
if (x == 20) {
    cout << "x是20";
}
```

**记忆口诀：`=` 赋值，`==` 比较。**

# 第3章：循环

## 3.1 理论：for 循环

**Lua：**
```lua
for i = 1, 5, 1 do
    print("第" .. i .. "秒")
end
```

**C++：**
```cpp
for (int i = 1; i <= 5; i++) {
    cout << "第" << i << "秒" << endl;
}
```

**逐段解释：**
```cpp
for (
    int i = 1;    // 第1段：初始化，只执行一次
    i <= 5;        // 第2段：条件判断，每次循环前检查
    i++            // 第3段：步进，每次循环后执行
) {
    // 循环体
}
```

**i++ 等价于 `i = i + 1`**

---

## 3.2 理论：while 循环

**适用：不知道次数，只知道退出条件**

```cpp
int level = 1;
int exp_needed = 1000;
int current_exp = 0;

while (current_exp < exp_needed) {
    current_exp += 80;
    level++;
}
cout << "升至" << level << "级" << endl;
```

---

## 3.3 理论：break 和 continue

| 命令 | 作用 |
|------|------|
| `break` | 跳出整个循环 |
| `continue` | 跳过本次循环，继续下一次 |

## 【实战3-1】战斗伤害计算

**任务：** 输入敌人攻击力和攻击间隔，计算30秒内总伤害

```cpp
#include <iostream>
using namespace std;

int main() {
    double player_hp;
    int attack;
    int interval;

    cout << "玩家HP: ";
    cin >> player_hp;
    cout << "敌人攻击力: ";
    cin >> attack;
    cout << "攻击间隔(秒): ";
    cin >> interval;

    double current_hp = player_hp;
    int total_damage = 0;
    int count = 0;

    for (int t = 0; t < 30; t += interval) {
        current_hp -= attack;
        total_damage += attack;
        count++;

        if (current_hp <= 0) {
            cout << "第" << t << "秒: 玩家HP归零，死亡" << endl;
            break;
        }
        cout << "第" << t << "秒: 受到" << attack
             << "伤害，剩余HP:" << current_hp << endl;
    }

    cout << "\n总攻击次数:" << count << endl;
    cout << "总伤害:" << total_damage << endl;
    cout << "剩余HP:" << (current_hp > 0 ? current_hp : 0) << endl;

    return 0;
}
```

## 【实战3-2】完整回合制战斗

**任务：** 玩家 vs 敌人，轮流攻击，一方倒下为止

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string player_name = "战士";
    double player_hp = 100;
    int player_atk = 25;

    string enemy_name = "哥布林";
    double enemy_hp = 80;
    int enemy_atk = 15;

    cout << "===== 战斗开始 =====" << endl;
    cout << player_name << " HP:" << player_hp << " 攻击:" << player_atk << endl;
    cout << enemy_name << " HP:" << enemy_hp << " 攻击:" << enemy_atk << endl;
    cout << endl;

    int round = 1;

    while (player_hp > 0 && enemy_hp > 0) {
        cout << "===== 第" << round << "回合 =====" << endl;

        // 玩家攻击
        enemy_hp -= player_atk;
        cout << player_name << " 攻击 " << enemy_name
             << "，造成" << player_atk << "伤害，敌人HP:"
             << (enemy_hp > 0 ? enemy_hp : 0) << endl;

        if (enemy_hp <= 0) break;

        // 敌人攻击
        player_hp -= enemy_atk;
        cout << enemy_name << " 攻击 " << player_name
             << "，造成" << enemy_atk << "伤害，玩家HP:"
             << (player_hp > 0 ? player_hp : 0) << endl;

        cout << endl;
        round++;
    }

    cout << "===== 战斗结束 =====" << endl;
    if (player_hp > 0) {
        cout << "【胜利】" << player_name << "击败了" << enemy_name << endl;
        cout << "剩余HP:" << player_hp << endl;
    } else {
        cout << "【失败】" << player_name << "被" << enemy_name << "击败" << endl;
    }

    return 0;
}
```


# 第5章：函数

## 5.1 理论：函数是什么

**Lua 的函数：**
```lua
function add_hp(hp, amount)
    return hp + amount
end
```

**C++ 的函数：**
```cpp
int add_hp(int hp, int amount) {
    return hp + amount;
}
```

**关键区别：**
- C++ 要声明返回类型（这里是 `int`）
- C++ 参数要声明类型（`int hp`）
- Lua 可以返回任意类型，C++ 必须在声明时指定类型

---

## 5.2 理论：函数参数传递（值传递）

```cpp
void add_hp(int hp, int amount) {
    hp = hp + amount;  // 改的是副本，不影响原值
}

int main() {
    int player_hp = 100;
    add_hp(player_hp, 30);
    cout << player_hp;  // 输出 100，没变！
}
```

这是值传递——函数收到的是变量的副本，不是原件。

## 6.2 理论：new 和 delete


## 【实战6-1】用 struct 管理玩家数据

```cpp
#include <iostream>
#include <string>
using namespace std;

struct Player {
    string name;
    int level;
    double hp;
    double max_hp;
    int attack;
    int defense;
};

// 显示玩家状态
void show_player(const Player& p) {
    cout << "===== " << p.name << " =====" << endl;
    cout << "等级: " << p.level << endl;
    cout << "HP: " << p.hp << "/" << p.max_hp << endl;
    cout << "攻击: " << p.attack << " 防御: " << p.defense << endl;
}

// 扣血
void take_damage(Player& p, double dmg) {
    double actual = dmg - p.defense * 0.5;
    if (actual < 1) actual = 1;
    p.hp -= actual;
    if (p.hp < 0) p.hp = 0;
    cout << p.name << "受到" << actual << "伤害，剩余HP:" << p.hp << endl;
}

// 加血
void heal(Player& p, double amount) {
    p.hp += amount;
    if (p.hp > p.max_hp) p.hp = p.max_hp;
    cout << p.name << "恢复" << amount << "HP，当前HP:" << p.hp << endl;
}

int main() {
    // 创建玩家
    Player player;
    player.name = "战士";
    player.level = 30;
    player.hp = 100;
    player.max_hp = 100;
    player.attack = 25;
    player.defense = 10;

    // 创建敌人
    Player enemy;
    enemy.name = "哥布林";
    enemy.level = 20;
    enemy.hp = 80;
    enemy.max_hp = 80;
    enemy.attack = 15;
    enemy.defense = 5;

    // 战斗
    cout << "===== 战斗开始 =====" << endl;
    show_player(player);
    show_player(enemy);
    cout << endl;

    // 玩家攻击敌人
    take_damage(enemy, player.attack);
    cout << endl;

    // 敌人攻击玩家
    take_damage(player, enemy.attack);
    cout << endl;

    // 玩家喝药水
    heal(player, 30);

    return 0;
}
```

```
战士受到25伤害，剩余HP:55
```

## 【实战6-2】动态创建玩家对象

**任务：** 根据用户输入，动态创建 N 个玩家并管理

```cpp
#include <iostream>
#include <string>
using namespace std;

struct Player {
    string name;
    int level;
    double hp;
};

int main() {
    int count;
    cout << "创建几个玩家: ";
    cin >> count;

    // 动态创建数组
    Player* players = new Player[count];

    // 输入每个玩家信息
    for (int i = 0; i < count; i++) {
        cout << "玩家" << (i+1) << "的名字: ";
        cin >> players[i].name;
        cout << "玩家" << (i+1) << "的等级: ";
        cin >> players[i].level;
        players[i].hp = 100;
    }

    // 打印所有玩家
    cout << "\n===== 玩家列表 =====" << endl;
    for (int i = 0; i < count; i++) {
        cout << i+1 << ". " << players[i].name
             << " 等级" << players[i].level
             << " HP" << players[i].hp << endl;
    }

    // 记得释放内存！
    delete[] players;

    return 0;
}
```

**注意：`new` 必须配对 `delete[]`，否则内存泄漏。**
