# 第1章：变量与数据类型

## 1.1 变量

**Lua 的变量（类型随时可变）：**
```lua
name = "战士"
name = 100  -- 合法，Lua 自动识别类型
```

**C++ 的变量（先声明类型，类型固定）：**
```cpp
string name = "战士";
int level = 30;
double hp = 100.0;

name = 100;  // 错误！编译时报错
```

---

## 1.2 常用数据类型

| 类型 | 含义 | 举例 |
|------|------|------|
| `int` | 整数 | `30`, `-10` |
| `double` | 浮点数 | `100.5` |
| `bool` | 布尔 | `true`, `false` |
| `string` | 字符串 | `"战士"`（需 `#include <string>`）|

---

## 1.3 cout 输出、cin 输入

```cpp
cout << "等级:" << level << endl;  // 打印
cin >> name >> level;              // 输入，多个连续读
```

---

## 【实战】玩家信息管理

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    int level;
    double hp;

    cin >> name >> level >> hp;

    double power = level * 10 + hp * 0.5;
    cout << name << " 战斗力:" << power << endl;

    return 0;
}
```
