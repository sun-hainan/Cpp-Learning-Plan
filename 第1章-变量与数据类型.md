# 第1章：变量与数据类型

## 1.1 C++ 是什么？和 Lua 有什么区别？

**你写传奇脚本用的是 Lua，C++ 是另一个层面的工具。**

| 对比 | Lua | C++ |
|------|-----|-----|
| 运行方式 | 边解释边执行 | 先编译成机器码再执行 |
| 速度 | 较慢 | 极快 |
| 内存管理 | 自动GC | 手动管理 |
| 类型检查 | 动态（自动推断） | 静态（先声明再使用） |

**学 C++ 的理由：游戏引擎核心都是 C++，懂了底层，Lua 脚本写得更好。**

## 1.2 第一个程序：编译和运行

**C++ 程序分两步：**

```
hello.cpp  →  编译  →  hello.exe  →  运行
```

**在你的电脑上编译运行：**

```powershell
powershell -ExecutionPolicy Bypass -File C:\cpp\compile.ps1 C:\cpp\hello.cpp
```

## 1.3 HelloWorld 逐行解析

```cpp
#include <iostream>      // 引入输入输出工具
using namespace std;   // 使用标准命名空间

int main() {            // 程序入口
    cout << "Hello, World!" << endl;
    return 0;           // 正常结束
}
```

- `#include <iostream>`：把工具箱复制进来，提供 `cout`、`cin`
- `using namespace std;`：`cout` 不用写成 `std::cout`
- `int main() { }`：主函数，程序从这里开始

## 1.4 变量：给数据起名字

**Lua 的变量（类型随时可变）：**
```lua
name = "战士"
name = 100  -- 合法，Lua 的变量可以随时改变类型
```

**C++ 的变量（声明后类型固定）：**
```cpp
string name = "战士";   // 字符串
int level = 30;          // 整数
double hp = 100.0;       // 浮点数
name = 100;  // 错误！编译时报错，name 是字符串类型，不能赋整数
```

**为什么要声明类型：**
- 告诉计算机分配多少内存
- 编译时就检查错误
- 编译器能优化，运行更快

## 1.5 常用数据类型

| 类型 | 含义 | 举例 |
|------|------|------|
| `int` | 整数 | `30`, `-10`, `0` |
| `double` | 浮点数（小数） | `100.5`, `3.14` |
| `bool` | 布尔 | `true`, `false` |
| `string` | 字符串 | `"战士"`（需 `#include <string>`）|

**和 Lua 的对应：**

```
Lua           C++
─────────────────────
number        int / double
string        string
boolean       bool
table         数组+结构体+map
```

## 【实战】玩家信息管理

**任务：** 输入玩家名字、等级、HP，计算并打印战斗力

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    int level;
    double hp;
    int attack;
    int defense;

    cout << "===== 创建角色 =====" << endl;
    cin >> name >> level >> hp >> attack >> defense;

    double power = attack * 2 + defense + level * 5;
    cout << name << " 战斗力: " << power << endl;

    return 0;
}
```

**运行示例：**
```
===== 创建角色 =====
战士 30 100 50 20
战士 战斗力: 270
```

**用到的知识点：string、int、double、cout、cin，全部是本章内容。**
