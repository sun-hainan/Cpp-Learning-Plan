## 2.1 if / else：玩家状态判断

**在 Lua 里写过无数次的逻辑：**
```lua
if player_hp <= 0 then
    print("玩家死亡")
elseif player_hp < 30 then
    print("危险！血量过低")
else
    print("状态正常")
end
```

**C++ 的写法：**
```cpp
if (player_hp <= 0) {
    cout << "玩家死亡" << endl;
} else if (player_hp < 30) {
    cout << "危险！血量过低" << endl;
} else {
    cout << "状态正常" << endl;
}
```

**三个区别：**
| 区别 | Lua | C++ |
|------|-----|-----|
| 条件括号 | 不需要 | 必须加 `( )` |
| 代码块 | `then ... end` | `{ }` |
| 否则如果 | `elseif` | `else if`（两个词） |

## 2.2 if 的坑：赋值而不是比较

**C++ 特有的危险错误：**
```cpp
int x = 10;

// 错误：= 是赋值，条件永远为 true
if (x = 20) {
    cout << "x是20";  // 这行永远会执行
}

// 正确：== 才是比较
if (x == 20) {
    cout << "x是20";
}
```

**记忆口诀：**
```
=  是赋值，把右边的值放进左边的盒子
== 是比较，问左右两边是不是相等
```

## 【实战 3-4A】玩家血量状态判断

**任务：** 输入玩家血量，打印状态（满血/正常/危险/死亡）

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    double hp, max_hp;

    cout << "===== 玩家血量检测 =====" << endl;
    cout << "最大HP: ";
    cin >> max_hp;
    cout << "当前HP: ";
    cin >> hp;

    double hp_percent = hp / max_hp * 100;

    if (hp <= 0) {
        cout << "[死亡] HP为0，无法战斗" << endl;

        cout << "[危险] HP仅剩" << hp << "，建议使用药水" << endl;
    } else if (hp_percent <= 70) {
        cout << "[一般] HP剩余" << hp << "，注意恢复" << endl;
    } else {
        cout << "[满血] HP状态良好：" << hp << endl;
    }

    return 0;
}
```

**运行示例：**
```
===== 玩家血量检测 =====
最大HP: 100
当前HP: 25
[危险] HP仅剩25，建议使用药水
```

## 2.3 for 循环：已知次数的重复

**Lua：**
```lua
for i = 1, 5, 1 do
    print("第" .. i .. "秒")
end
```

**C++：**
```cpp
for (int i = 1; i <= 5; i++) {
    cout << "第" << i << "秒" << endl;
}
```

**逐段解释：**
```cpp
for (
    int i = 1;    // 第1段：初始化（只执行一次）
    i <= 5;        // 第2段：条件判断（每次循环前检查）
    i++            // 第3段：步进（每次循环后执行）
) {
    // 循环体
}
```

**i++ 等价于：**
```cpp
i = i + 1;
i += 1;
// 都是把 i 增加 1
```

## 【实战 3-4B】战斗伤害计算（循环扣血）

**任务：** 输入敌人攻击力，计算5秒内对玩家造成的总伤害

```cpp
#include <iostream>
using namespace std;

int main() {
    double player_hp;
    int enemy_attack;
    int attack_interval;  // 攻击间隔（秒）

    cout << "===== 战斗伤害计算 =====" << endl;
    cout << "玩家HP: ";
    cin >> player_hp;
    cout << "敌人攻击力: ";
    cin >> enemy_attack;
    cout << "攻击间隔（秒）: ";
    cin >> attack_interval;

    double current_hp = player_hp;
    int total_damage = 0;
    int attack_count = 0;

    cout << "\n===== 战斗过程 =====" << endl;

    for (int second = 0; second < 30; second += attack_interval) {
        attack_count++;
        current_hp -= enemy_attack;
        total_damage += enemy_attack;

        if (current_hp <= 0) {
            cout << "第" << second << "秒: 玩家受到" << enemy_attack
                 << "点伤害，HP归零，死亡！" << endl;
            break;
        }

        cout << "第" << second << "秒: 玩家受到" << enemy_attack
             << "点伤害，剩余HP:" << current_hp << endl;
    }

    cout << "\n===== 战斗结算 =====" << endl;
    cout << "攻击次数: " << attack_count << endl;
    cout << "总伤害: " << total_damage << endl;
    cout << "剩余HP: " << (current_hp > 0 ? current_hp : 0) << endl;

    return 0;
}
```

**运行示例：**
```

...
```

## 2.4 while 循环：不知道次数的重复

**适用场景：** 玩家不退出就一直打，直到血量归零或手动停止

```cpp
// 模拟一场战斗，直到一方倒下
double player_hp = 100;
double enemy_hp = 80;
int round = 0;

while (player_hp > 0 && enemy_hp > 0) {
    round++;
    // 玩家攻击
    enemy_hp -= 25;
    cout << "第" << round << "回合: 玩家攻击，敌人HP:" << enemy_hp << endl;

    if (enemy_hp <= 0) break;  // 敌人倒下，战斗结束

    // 敌人反击
    player_hp -= 15;
    cout << "第" << round << "回合: 敌人攻击，玩家HP:" << player_hp << endl;
}

if (player_hp > 0) {
    cout << "玩家获胜！" << endl;
} else {
    cout << "玩家失败..." << endl;
}
```

**for 和 while 的选择：**
| 场景 | 用哪个 |
|------|--------|
| 知道循环几次 | `for` |
| 不知道几次，只知道退出条件 | `while` |

## 【实战 3-4C】玩家升级经验计算

**任务：** 输入升级所需经验和每只怪物的经验，计算杀多少只才能升级

```cpp
#include <iostream>
using namespace std;

int main() {
    int exp_needed;    // 升级所需经验
    int exp_per_monster;  // 每只怪物给的经验

    cout << "===== 升级经验计算 =====" << endl;
    cout << "升级所需经验: ";
    cin >> exp_needed;
    cout << "每只怪物经验: ";
    cin >> exp_per_monster;

    int current_exp = 0;
    int monster_killed = 0;

    while (current_exp < exp_needed) {
        current_exp += exp_per_monster;
        monster_killed++;
    }

    cout << "需要击杀 " << monster_killed << " 只怪物才能升级" << endl;
    cout << "总获得经验: " << current_exp << endl;

    return 0;
}
```

**运行示例：**
```
===== 升级经验计算 =====
升级所需经验: 1000
每只怪物经验: 85
需要击杀 12 只怪物才能升级
总获得经验: 1020
```

## 2.5 break 和 continue：控制循环

| 命令 | 作用 | 类比 |
|------|------|------|
| `break` | 跳出整个循环 | 接到老板电话，立刻停止购物离开 |
| `continue` | 跳过本次循环，继续下一次 | 跳过今天的跑步，明天继续 |

**break 的例子：找到第一个可击败的敌人**
```cpp
int enemies[] = {50, 80, 30, 60, 25};
int player_attack = 40;

for (int i = 0; i < 5; i++) {
    if (enemies[i] <= player_attack) {
        cout << "找到可击败的敌人，索引:" << i
             << "，敌人HP:" << enemies[i] << endl;
        break;  // 找到第一个就停止
    }
}
```

**continue 的例子：跳过无法击败的敌人**
```cpp
for (int i = 0; i < 5; i++) {
    if (enemies[i] > player_attack) {
        continue;  // 打不过，跳过这个，继续看下一个
    }
    cout << "可以击败敌人，敌人HP:" << enemies[i] << endl;
}
```

## 【实战 3-4D】完整战斗系统

**任务：** 写一个完整的回合制战斗，支持：
- 玩家和敌人各有 HP、攻击力
- 轮流攻击，直到一方倒下
- 显示每回合的过程
- 战斗结束后显示结果

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // ========== 初始化战斗角色 ==========
    string player_name = "战士";
    double player_hp = 100;
    double player_max_hp = 100;
    int player_attack = 25;

    string enemy_name = "哥布林";
    double enemy_hp = 80;
    double enemy_max_hp = 80;
    int enemy_attack = 15;

    // ========== 战斗开始 ==========
    cout << "===== 战斗开始 =====" << endl;
    cout << player_name << " HP:" << player_hp << " 攻击:" << player_attack << endl;
    cout << enemy_name << " HP:" << enemy_hp << " 攻击:" << enemy_attack << endl;
    cout << endl;

    int round = 1;

    while (player_hp > 0 && enemy_hp > 0) {
        cout << "===== 第 " << round << " 回合 =====" << endl;

        // ---------- 玩家回合 ----------
        enemy_hp -= player_attack;
        cout << player_name << " 攻击 " << enemy_name
             << "，造成 " << player_attack << " 点伤害" << endl;
        cout << enemy_name << " 剩余HP: " << (enemy_hp > 0 ? enemy_hp : 0) << endl;

        if (enemy_hp <= 0) break;  // 敌人倒下，战斗结束

        // ---------- 敌人回合 ----------
        player_hp -= enemy_attack;
        cout << enemy_name << " 攻击 " << player_name
             << "，造成 " << enemy_attack << " 点伤害" << endl;
        cout << player_name << " 剩余HP: " << (player_hp > 0 ? player_hp : 0) << endl;

        if (player_hp <= 0) break;  // 玩家倒下，战斗结束

        cout << endl;
        round++;
    }

    // ========== 战斗结果 ==========
    cout << "\n===== 战斗结束 =====" << endl;
    if (player_hp > 0) {
        cout << "【胜利】" << player_name << "击败了" << enemy_name << "！" << endl;
        cout << "剩余HP: " << player_hp << endl;
    } else {
        cout << "【失败】" << player_name << "被" << enemy_name << "击败了..." << endl;
    }

    return 0;
}
```

**运行示例：**
```
===== 战斗开始 =====
战士 HP:100 攻击:25
哥布林 HP:80 攻击:15

===== 第 1 回合 =====
战士 攻击 哥布林，造成 25 点伤害
哥布林 剩余HP: 55
哥布林 攻击 战士，造成 15 点伤害
战士 剩余HP: 85

===== 第 2 回合 =====
战士 攻击 哥布林，造成 25 点伤害
哥布林 剩余HP: 30
哥布林 攻击 战士，造成 15 点伤害
战士 剩余HP: 70
...
===== 战斗结束 =====
【胜利】战士击败了哥布林！
剩余HP: 10
```

## 本周总结

| 概念 | 代码 | 含义 |
|------|------|------|
| if | `if (条件) { } else if (条件) { } else { }` | 条件分支 |
| else if | `else if`（两个词） | Lua 是 `elseif`（一个词） |
| for | `for (int i=0; i<5; i++) { }` | 已知次数的循环 |
| while | `while (条件) { }` | 不知道次数的循环 |
| break | 跳出整个循环 | 提前结束 |
| continue | 跳过本次循环 | 跳到下一次 |

- 战斗伤害计算（循环扣血）
- 升级经验计算（while 循环）
- 完整回合制战斗（if + for + while + break）

**这些都是你在 Lua 脚本里天天写的逻辑，用 C++ 实现后运行速度快 10 倍以上。**
